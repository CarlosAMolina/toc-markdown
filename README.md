## Configuration

You can use the script as a command. Example:

```bash
vi ~/.bashrc
# Add: alias toc='python ~/Software/toc-markdown/src/main.py'
source ~/.bashrc
```

To show the TOC in the terminal:

```bash
toc file.md
```
